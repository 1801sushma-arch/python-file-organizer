# python-file-organizer
a beginner - friendly python project that automatically organizes files into folders based on their file types.

Features
* Organizes Images, Documents, and Videos automatically
* Creates folders if they do not exist
* Uses Python modules like os and shutil
* Beginner-friendly project for learning file handling
  
Technologies Used: Python,os module,shutil module

File Types Supported
1.Images
.jpg
.png
2.Documents
.pdf
.txt
3.Videos
.mp4

How It Works
The program scans all files inside a folder and moves them into:Images folder,Documents folder,Videos folder

* Creating folders using Python
* Moving files automatically
* Using loops and conditions in real projects
